FG.Timepicker : Another Javascript Timepicker by François Gélinas (and Émilie B, and contributors)

Why?

So this is 2019 2020 2021 and the world is still changing but I still feel there is a missing usefull solution for a simple timepicker control.

I liked the first version i wrote some years ago but binding to a framework (jQuery) now feels like it is not the best way.

This implementation of the timepicker is completely vanilla and has no dependencies.
